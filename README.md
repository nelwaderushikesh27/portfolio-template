# 🎨 Portfolio Template

A modern, responsive portfolio website template built with HTML, CSS, and JavaScript.

## ✨ Features
- Fully responsive design
- Dark/Light mode toggle
- Smooth animations
- Contact form
- Project showcase
- Skills section
- Blog section ready
- SEO optimized
- Fast loading
- Mobile-first approach

## 🛠️ Tech Stack
- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Grid)
- **JavaScript** (ES6+)
- **No frameworks** - Pure vanilla!

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/nelwaderushikesh27/portfolio-template.git

# Open index.html in browser
open index.html
```

## 🎨 Customization

### Colors
Edit CSS variables in `css/style.css`:
```css
:root {
  --primary-color: #6366f1;
  --secondary-color: #8b5cf6;
  --accent-color: #06b6d4;
  --bg-color: #0f172a;
  --text-color: #e2e8f0;
}
```

### Content
Update your information in `index.html`:
- Name and title
- About description
- Projects
- Skills
- Contact info

## 📱 Responsive Breakpoints
- Mobile: < 480px
- Tablet: 481px - 768px
- Desktop: 769px - 1024px
- Large: > 1025px
